# 1 标题
CharacterGen: Efficient 3D Character Generation from Single Images with Multi-View Pose Calibration
角色生成：基于单张图像的高效3D角色生成与多视角姿态校准
# Abstract
在数字内容创作领域，从单张图像生成高质量的 3D 角色极具挑战性，特别是考虑到各种复杂的身体姿势、自遮挡以及姿势歧义等问题 。在本文中，我们提出了 CharacterGen，这是一个旨在高效生成 3D 角色的框架 。CharacterGen 引入了一种流水线式（streamlined）生成流程，并结合了图像驱动的多视角扩散模型 。该模型能够有效地将输入的各种姿势校准为规范姿势（canonical form），同时保留输入图像的关键属性，从而解决了由多样化姿势带来的挑战 。我们方法的另一个核心组件是基于 Transformer 的通用稀疏视角重建模型，它能够基于多视角图像创建精细的 3D 模型 。此外，我们还采用了纹理反向投影（texture-back-projection）策略来生成高质量的纹理贴图 。为了训练和评估我们的模型，我们策划了一个动漫角色数据集，其中包含在多种姿势和视角下渲染的角色 。通过定性和定量的实验评估，我们的方法表现出了卓越的性能，能够生成具有高质量形状和纹理的 3D 角色，并可直接用于后续的**绑定（rigging）和动画（animation）**工作流 。
# 1 Introduction
数字内容产业的快速演进使得创建高质量的 3D 内容成为电影、视频游戏、在线直播和虚拟现实 (VR) 等各个领域的关键环节 。虽然手动建模的 3D 内容可以达到极高的质量，但其所需的大量时间和劳动力投入构成了实质性的瓶颈 。为了解决这一问题，涌现了大量令人振奋的研究，专注于从单张图像生成 3D 模型 。这种方法显著降低了初学者的准入门槛，使 3D 内容创作更加大众化，并有望彻底改变该领域 。

3D 角色模型通常具有复杂的关节连接，这导致 2D 图像中频繁出现自遮挡现象，给重建、生成和动画带来了巨大挑战 。此外，这些角色可能会呈现各种身体姿势，其中一些姿势罕见且难以准确解读，导致了多样但极不均衡的数据域 。这些复杂性阻碍了此类模型的有效生成、绑定和动画制作 。因此，通用的 3D 生成技术和单视角 3D 重建方法往往难以提供理想的结果 。

先前研究探索了利用 3D 人体参数模型（如 SMPL）作为 3D 先验知识 。然而，这些方法主要针对写实的人体比例和相对紧身的衣物，限制了其适用性 。这种局限性在风格化角色的背景下尤为明显，这些角色以夸张的身体比例和复杂的服装设计著称，这对上述方法的适应性和有效性提出了挑战 。

在本文中，我们介绍了 CharacterGen，这是一种从单张图像生成规范姿势（canonical pose）下的 3D 角色的新方法 。我们的方法与以往方法的显著不同之处在于，它允许输入图像中存在任何身体姿势，并能输出一个整洁的 3D 角色模型 。

CharacterGen 的核心原则是在生成过程中同时实现身体姿势的规范化并生成一致的多视角图像 。这是通过将每种姿势转换为规范的“A-pose”（一种在 3D 角色建模中广泛采用的姿态）来实现的，同时确保多个视角之间的图像一致性 。这种双重方法有效解决了自遮挡和模糊的人体姿势问题，显著简化了后续的重建、绑定和动画阶段 。

我们的 3D 角色生成方法包含两个紧密相连的阶段：
多视角提升与姿势规范化：首先将单张图像提升到多个视点，同时将输入姿势规范化 。
3D 重建：随后使用该规范姿势重建 3D 角色 。

该方法得益于两个关键见解：首先，它融入了近期可控图像生成领域的成功技术 ；其次，它克服了 3D 角色稀疏视角重建的挑战 。通过聚焦于几何和纹理结构定义更清晰、且自遮挡最小的规范姿势，我们的方法简化了从有限视角重建几何和纹理的任务 。

第一阶段涉及到一个基于扩散的图像条件多视角生成模型，该模型能够将输入图像的全局和局部角色特征捕获并转换到规范姿势中 。第二阶段采用基于 Transformer 的通用稀疏视角重建模型，从第一阶段生成的图像中产生粗糙纹理的 3D 角色模型 。我们通过投影纹理映射和 Poisson 融合（Poisson Blending）进一步提升纹理分辨率，最终获得精细模型 。此外，生成规范姿势的角色也极大惠及了绑定和动画等下游应用 。

我们的整个生成过程耗时不到 1 分钟 。为了训练我们的流程，我们编译了一个包含 13,746 个风格化角色的多姿势、多视角数据集，称为 Anime3D 。
总而言之，本论文的主要贡献如下：
图像条件扩散模型：能从变化的输入姿势中，有效生成受控规范姿势下的多视角一致图像 。
流程化生成管线：结合了多视角图像生成的扩散模型和基于 Transformer 的重建模型，高效将单视角输入转化为详细的 3D 角色模型 。
**Anime3D 数据集**：包含 13,746 个动漫角色的策划数据集，为 3D 角色生成研究提供了多样化的训练和评估资源 。
# 2 Related Work
本节主要讨论基于扩散模型的 3D 物体和头像（Avatar）生成的相关工作 。由于篇幅限制，此处不再详细展开 3D 人体重建的相关研究 。此外，我们的 CharacterGen 还采用了基于 Transformer 的重建模型来实现高效的 3D 角色生成 。

## 2.1 基于扩散模型的 3D 物体生成 (Diffusion-Based 3D Object Generation)
在过去的一年里，扩散方法在引导 3D 物体生成任务中展现出了强大的能力 。
* 分数蒸馏采样 (SDS)：DreamFusion 和 SJC 的开创性工作利用分数蒸馏采样（SDS），从预训练的 2D 扩散模型中提供梯度引导，用于文本生成 3D 任务 。
* 高分辨率细化：Magic3D 和 Fantasia3D 利用隐式四面体场（implicit tetrahedral field），在细化阶段支持高分辨率渲染 。
* 变分分数蒸馏 (VSD)：ProlificDreamer 提出了 VSD，通过从 LoRA 网络中蒸馏梯度分数，以更好地学习 3D 物体的分布 。
* 多视角图像生成：Zero123 提出了一种新型扩散模型，能在给定相机姿态下生成与输入图像一致的多视角图像 。Magic123 则结合了 SDS 和 Zero123 的引导来生成 3D 物体，并采用重建损失（reconstruction loss）来增强正面纹理质量 。
* 视角一致性引导：MVDream 和 ImageDream 利用多视角扩散模型，在 3D 物体生成过程中提供高度一致性的引导 。SyncDreamer 利用 3D 感知注意力模块来实现同步的多视角图像生成 。
* 直接 3D 生成：其他一些工作采用 3D 数据训练扩散模型以直接生成 3D 物体，这种方式速度较快，但在输出多样性方面存在困难 。
## 2.2 3D 头像生成 (3D Avatar Generation)
利用 SMPL 和 SMPL-X 等强大的人体结构先验，可以在通用 3D 生成方法的基础上生成高质量的人体头像 。
* 人体先验引导：EVA3D 结合了 GAN 骨干网络和姿态引导采样方法；AvatarCLIP 则利用预训练的 CLIP 模型引导几何和颜色网络的优化 。
* 隐式几何初始化：DreamAvatar 和 AvatarCraft 利用 SMPL 初始化扩散引导生成过程中的隐式人体几何 。DreamHuman 采用 ImGHum 作为身体先验，并提出了一种聚焦渲染机制以更好地重建头像的细节几何 。
* 姿态与控制引导：DreamWaltz 利用 ControlNet 为微调动画表示提供姿态引导 。Avatar Verse 和 AvatarStudio 都在生成过程中利用 DensePose 引导的 ControlNet，以规避多面“雅努斯（Janus）”问题并支持局部几何优化 。
* 图像提示生成：TeCH 通过在输入图像上训练额外的 DreamBooth 模型来支持图像提示的头像生成 。TADA 直接蒸馏 2D 扩散模型，以优化 SMPL 人体网格的法线和位移 。
* 现有方法的局限性：大多数此类方法主要侧重于文本生成 3D 角色，无法利用图像提示，而这对于可控的角色生成至关重要 。此外，基于 DreamBooth 的方法由于过度拟合单张输入图像而导致严重的正面偏见，从而深受“雅努斯”问题的困扰 。
# 3 Method
我们现在阐述 CharacterGen 的整体框架，其目标是从任意姿势的 2D 图像中高效生成 A-pose（A 姿态）的 3D 角色 。第 3.1 节首先介绍我们的 Anime3D 数据集，展示我们如何组织数据以辅助扩散模型进行 3D 空间理解和角色姿势规范化 。第 3.2 节随后解释 CharacterGen 如何生成高度一致的多视角姿势规范化角色图像 。最后，第 3.3 节展示我们高效的 3D 重建流水线 。

## 3.1 Anime3D 数据集 (Anime3D Dataset) 
为了进一步提高扩散模型理解 3D 角色的能力并缓解“双面人（Janus）”问题，我们准备了包含 13,746 个风格化角色主体的 Anime3D 数据集 。
* 3.1.1 数据获取 (Data Acquisition)：现有的 Objaverse 或 OmniObject3D 等大型 3D 物体数据集不包含足够数量的 3D 风格化角色用于我们的训练 。受到 PAniC3D 的启发，我们首先从 VRoidHub 收集了近 14,500 个动漫角色，随后移除了非类人数据，最终保留了 13,746 个角色模型 。
* 3.1.2 数据处理 (Data Processing)：我们需要将所有物体渲染成图像格式以微调 2D 扩散模型 。我们利用 three-vrm 框架进行渲染 。
* 我们首先获取“A-pose”角色和姿态化的角色，以生成规范姿势和随机姿势的图像对 。
* 对于 A-pose 角色，我们将左右臂在 Z 轴上的关节旋转设置为 45°，左右大腿在 Z 轴上的旋转设置为 6° 。所有其他关节参数保持不变 。
* 对于姿态化（posed）角色的设置，我们从 Mixamo 下载了 10 个人体骨骼动画（包括坐下、唱歌、行走等），随机选择帧并将运动应用到 VRM 模型上 。
* 此外，我们还随机化了嘴部和眼部关节，以生成各种面部表情（如眨眼） 。
* 我们将角色模型的包围盒归一化为 $[-0.5, 0.5]^3$ 。
* 训练过程中，我们将四张 A-pose 图像与一张姿态化图像作为一组配对，因为正交视角的四张图像已包含 3D 角色的充分外观信息 。
* 物体渲染的方位角（azimuth）设置为 {0°, 90°, 180°, 270°}，仰角（elevation）为 0° 。
## 3.2 多视角图像生成与姿势规范化 (Multi-view Image Generation and Pose Canonicalization) 
我们现在考虑如何从给定的角色图像生成高度一致的多视角图像 。整体框架如图 3 所示 。
* 3.2.1 IDUNet：旨在保留原始姿态图像的充分特征，并确保四个生成视角之间的高度一致性 。以往的 IP-Adapter 仅利用全局 CLIP 嵌入，会导致像素级细节丢失 。我们提出的 IDUNet 引入了像素级引导，通过 VAE 编码无噪声的输入图像，并在潜空间令牌（latent tokens）与条件图像令牌之间利用**交叉注意力（cross-attention）**实现局部特征交互 。
* 3.2.2 多视角 UNet (Multi-view UNet)：目标是从单张姿态输入图像生成外观高度一致的多视角 A-pose 图像 。该模型包含空间自注意力模块和交叉注意力模块，通过同时处理四个视角的噪声潜变量并结合相机视角嵌入，捕捉跨视角的全局关系 。
* 3.2.3 姿势规范化 (Pose Canonicalization)：为了在生成过程中实现姿势规范化，我们联合训练两个 UNet 。我们采用 OpenPose 预测姿势嵌入作为额外条件，将其直接添加到潜变量噪声中，辅助模型学习关节与生成布局之间的关系 。
## 3.3 3D 角色生成 (3D Character Generation) 
我们采用由粗到精（coarse-to-fine）的过程进行 3D 重建 。
1. 粗纹理角色重建：受到 LRM 的启发，我们利用深度 Transformer 网络从四视角图像中高效重建角色 。
* 我们首先在 Objaverse 数据集上预训练，然后利用 Anime3D 数据集进行微调，以引入更多人体结构先验 。
* 我们采用两阶段微调策略：第一阶段使用三平面（triplane）NeRF 表示建立几何和外观；第二阶段将解码器修改为预测带符号距离函数 (SDFs)，以获得更平滑精确的表面 。
2. 3D 角色细化 (Refinement)：虽然可以通过 DMTet 提取网格，但该过程会丢失部分外观信息 。
* 我们利用生成的四视角图像通过纹理反向投影策略来增强纹理质量 。
* 为了消除轮廓周围的噪声，我们执行深度测试并计算相机视角与法线的内积，剔除遮挡和边缘的纹理像素（texels） 。
* 最后，利用 Poisson 融合 聚合投影纹理以减少接缝 。


# 4 Experiments
## 4.1 实现细节 (Implementation Details) 
* 我们将 Anime3D 数据集按 50:1 的比例划分为训练集和测试集 。
* 我们使用 Stable Diffusion 2.1 作为 IDUNet 和多视角 UNet 的基础模型 。
* 训练过程在 8 台 NVIDIA A800 GPU 上进行：首先在 $512\times512$ 分辨率图像上训练 3 天，随后在 $768\times512$ 分辨率上再训练 2 天 。
* 在每个训练步中，我们联合训练 IDUNet 和多视角 UNet 。为了加强扩散模型对 3D 角色的空间理解，我们以 0.8 的概率采样方位角为 {0°, 90°, 180°, 270°} 的图像组 。
* 微调基于 Transformer 的重建模型时，首先在 NeRF 表示上训练 50 个 epoch，随后在 SDF 上训练 30 个 epoch，此过程在 8 台 A800 GPU 上耗时 1 天 。
* 整个生成流水线（包括图像生成、网格重建和纹理细化）可在单张 GPU 上运行 。
## 4.2 结果与对比 (Results and Comparison) 
4.2.1 2D 多视角生成 (2D Multi-view Generation) 
* 我们将 CharacterGen 与 Zero123 和 SyncDreamer 进行对比 。
* 实验结果显示，在面对困难的身体姿势时，基线方法难以保留生成的几何和外观信息，而 CharacterGen 能够熟练地执行规范姿势校准 。
* 定量评估（见表 1）证明了 CharacterGen 在生成质量以及与多视角扩散模型一致性方面的优越性 。
## 4.2.2 3D 角色生成 (3D Character Generation) 
* 我们将结果与 ImageDream、Magic123 和 TeCH 等图像提示 3D 生成方法进行可视化对比（见图 6） 。
* 我们选择 倒角距离 (Chamfer Distance, CD) 作为评估生成网格几何质量的指标 。
* 得益于稳健的四视角重建机制，CharacterGen 有效避免了“雅努斯 (Janus)”问题（即多面问题），并能为未见过的身体部分提供满意的外观 。
* 其他方法生成的角色往往存在网格面内聚问题，导致难以进行绑定和动画制作，而 CharacterGen 生成的规范姿势网格便于下游应用 。
## 4.2.3 与 IP-Adapter 的对比 (Comparison with IP-Adapter) 
* 由于 IP-Adapter 缺乏 Stable Diffusion 2.1 的预训练模型，我们训练了一个 SDXL 基础模型进行对比 。
* 结果表明，IP-Adapter 无法保留详细的外观信息，且可能无法生成正确的角色布局，而 CharacterGen 通过 IDUNet 有效生成了高度一致的图像 。
## 4.2.4 生成速度 (Generation Speed) 
如表 2 所示，我们的方法显著快于其他替代方案 。CharacterGen 生成单个 3D 角色的时间约为 1 分钟，而 Magic123 需要 70 分钟，ImageDream 需要 45 分钟 。
## 4.3 用户研究 (User Study) 
* 我们邀请了 21 名志愿者 评估风格一致性、空间一致性、几何质量和纹理质量 。
* 如表 3 所示，CharacterGen 在 2D 和 3D 生成任务中均获得了显著更高的偏好 。
* 我们还采用了 CLIP score 作为客观指标，结果显示 CharacterGen 在 2D 和 3D 任务中均获得了最高分，验证了 IDUNet 强大的外观建模能力 。
## 4.4 消融实验 (Ablation Study) 
* 4.4.1 IDUNet：如果冻结 IDUNet 仅使用预训练模型，生成的图像将无法保留输入图像的充分特征 。这证明了联合微调 IDUNet 对于提取服装和面部细节的必要性 。
* 4.4.2 姿势嵌入网络 (Pose Embedding Network)：该网络对于保持角色布局至关重要 。缺乏布局引导会导致角色在图像中位置偏移，并产生不一致的服装部分，从而损害后续的 3D 重建 。
## 4.5 应用 (Applications) 
* CharacterGen 简化了绑定过程，我们使用 AccuRig 自动为生成的网格进行绑定 。
* 绑定的 3D 角色可直接在 Warudo 等软件中作为动画资产使用（见图 10） 。
* 对比实验（见图 11）显示，我们的 A-pose 角色能成功进行动画制作，而其他非规范姿势生成的角色会出现严重的网格畸变 。

# 5 Conclusion
虽然我们的方法可以从任意姿势的单张输入图像生成 3D 角色，但仍存在某些局限性 。

极端姿势下的信息丢失：在四视角 A-pose 图像生成阶段，当角色处于极端姿势或从非常规视角进行渲染时，我们的方法可能无法保留足够的信息 。

未来工作方向 - 纹理增强：作为未来的研究方向，在纹理细化阶段集成额外的非写实渲染（NPR）技术，可能会进一步提升生成角色的纹理质量 。

几何质量优化：此外，利用我们训练好的多视角 UNet 结构，未来可能通过引入分数蒸馏采样（SDS）优化方法，来实现具有更优几何质量的 3D 角色生成 。

本文提出了 CharacterGen，这是一个新颖且高效的图像提示（image-prompt）3D 角色生成框架 。

数据集贡献：我们编译了一个全新的多姿势、风格化角色数据集 Anime3D 来训练我们的流水线 。

核心架构设计：我们的设计包括 IDUNet，它从输入的条件图像中提取片元级（patch-level）特征，用以引导多视角 A-pose 角色图像的生成 。

重建与细化策略：随后，我们利用基于 Transformer 的网络重建 3D 角色网格，并提出利用纹理反向投影细化策略来进一步改善重建角色的外观 。

应用价值：实验表明，CharacterGen 能够高效生成高质量的 3D 角色，适用于多种下游应用场景 。

# 审稿
## 总体评价
该研究提出了一种创新的“姿态规范化（Pose Canonicalization）”思路，通过 IDUNet 提取像素级特征，并结合 Transformer 重建模型，实现了从单张任意姿势图像到 A-pose 3D 模型的快速转化 。其核心贡献在于将复杂的 3D 生成任务简化为规范化姿态下的重建任务，显著提升了下游骨骼绑定与动画的兼容性 。
## 优点（Strengths）
* 工业级应用的前瞻性： 论文敏锐地捕捉到非规范姿势（Non-canonical pose）生成 3D 模型时产生的“网格粘连（Mesh Cohesion）”痛点，通过强制生成 A-pose，确保了模型在动画变形时的结构完整性 。
* 高效的端到端管线： 整个推理流程在单卡上仅需不到 1 分钟，相比 Magic123（70分钟）和 ImageDream（45分钟）具有量级上的速度优势，具备实时或近实时生成的潜力 。
* IDUNet 像素级控制机制： 不同于传统 IP-Adapter 仅依赖全局 CLIP 嵌入，作者提出的 IDUNet 通过交叉注意力机制引入了像素级引导，有效解决了纹理细节丢失和多视角不一致的问题 。
* 高质量数据集贡献： 策划并发布了包含 13,746 个风格化角色的 Anime3D 数据集，填补了现有大规模 3D 物体数据集（如 Objaverse）在二次元/风格化类人角色领域的空白 。
## 核心缺陷（Weaknesses）
几何表现力受限： 尽管采用了 SDF 预测以获得更平滑的表面，但 3D 重建部分本质上仍依赖于三平面（Triplane）表示 。这种方法在处理极其复杂的服装褶皱或精细附件时，几何精度往往不如基于 SDS（分数蒸馏采样）的优化方法 。
数据域偏置（Domain Bias）： 模型高度依赖 Anime3D 数据集 。这意味着对于写实风格人体或非主流美术风格，其姿态规范化与纹理补全的泛化能力可能显著下降，论文缺乏对异质数据分布的鲁棒性测试。
极端姿态下的语义丢失： 当输入图像处于极端透视或严重自遮挡的“Tricky Poses”时，IDUNet 提取的局部特征可能不足以支撑背部及侧面纹理的合理推断，导致生成模型在这些视角下出现模糊或错误的语义表达 。
纹理反向投影的局限： 采用的 Poisson Blending 虽然能缓解接缝问题，但在多视角投影重叠区域，若生成的 2D 图像存在像素偏移，最终生成的纹理贴图仍可能出现重影或锯齿现象 。

## 后续研究方向（Future Directions）
引入 SDS 细化几何： 正如作者在讨论中所述，可以尝试将训练好的多视角一致性 UNet 作为先验，引入 SDS 优化过程，以牺牲部分时间成本为代价，获取具有影视级精细度的几何结构 。
集成非写实渲染（NPR）算子： 针对动漫风格，在纹理细化阶段引入 NPR 技术（如卡通着色、描边算法），可以进一步增强模型在渲染管线中的视觉表现力 。
多模态一致性增强： 探索将文本提示词（Text Prompt）与图像特征进行更深层的语义对齐，利用 LLM 的常识推理能力来辅助推断被遮挡部分的物理属性（如裙子背后的材质或配饰结构）。
动态骨骼感知的生成： 考虑将角色骨骼布局（Bone Layout）直接融入生成逻辑，而不仅仅是作为外部的 OpenPose 引导，从而实现从单图直接生成带权重（Skinning Weights）的 3D 资产。
