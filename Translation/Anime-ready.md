# 1. 标题+作者
ANIME-READY：通过身体对齐的组件级衣物建模，实现可控的 3D 动漫角色生成
香港大学 南京大学
# 2. 摘要
3D动漫角色生成在数字娱乐领域，包括动画制作、虚拟现实、游戏和虚拟网红等领域，变得日益重要。与写实的人体建模不同，动漫风格的角色需要夸张的比例、风格化的表面细节以及艺术上一致的服装，这给自动化3D生成带来了独特的挑战。以往的3D动漫角色生成方法通常存在网格质量低、纹理模糊等问题，而且通常无法提供相应的骨骼，限制了其在动画制作中的应用。本文提出了一种用于高质量3D动漫角色生成的新框架，该框架结合了蒙皮多人线性（SMPL）模型的强大表现力和精确的服装生成技术，克服了上述局限性。我们的方法扩展了Anime-SMPL模型，使其能够更好地捕捉动漫角色的独特特征，实现统一的骨骼生成和基于混合变形的面部表情控制。最终生成了具有丰富面部表情、身体和服装的、完全可用于动画制作的3D角色。为了完善人体模型，我们引入了一种基于身体的组件式服装生成流程（包括发型、上衣、下装和配饰），该流程将服装建模为与身体几何形状对齐的结构化组件。此外，我们的方法能够生成高质量的皮肤和面部纹理，以及精细的服装纹理，从而提升生成角色的视觉保真度。实验结果表明，我们的框架在网格质量、纹理清晰度和服装与身体的对齐度方面均显著优于基线方法，使其适用于动漫内容创作和互动媒体等多种应用场景。

# 3. 引言
3D 动漫角色的需求在游戏、虚拟偶像、增强现实/虚拟现实（AR/VR）等多个行业中呈现爆发式增长。然而，手动创作高质量、具备动画能力的 3D 角色需要大量的人力和高度专业化的技能。虽然近年来在自动化 3D 生成方面取得了显著进展，但在追求“动画就绪（Animation-Ready）”和“精确可控”的 3D 动漫角色方面，仍然存在重大挑战。

现有的方法通常采用通用的 3D 生成技术（如生成网格或点云），但在处理动漫风格时常面临以下局限：首先，动漫角色的比例十分特殊且夸张（如极大的眼睛和独特的肢体长度），现有的 3D 人体模型难以适配；其次，衣物与身体的对齐质量较差，导致在动画过程中容易出现严重的穿模（Interpenetration）现象；最后，生成的模型通常缺乏高质量的内骨架和面部表情参数（Blendshapes），使其无法直接投入生产环境。

为了填补这一空白，我们提出了 ANIME-READY。其核心思想是将统计人体模型（SMPL）的严谨性与组件化建模的灵活性相结合。我们通过大规模动漫数据微调，将原有的 SMPL 进化为 Anime-SMPL，不仅能准确拟合动漫风格，还自带骨骼。同时，我们通过一种“身体对齐”的组件建模流水线，将衣物拆分为发型、上衣、裙裤等独立层次进行生成，确保了极高的网格拓扑质量和纹理清晰度。

# 4. 结论
在本文中，我们介绍了一个高质量的 3D 动漫角色生成框架。通过引入 Anime-SMPL，我们成功弥合了 2D 动漫图像与具备动画能力的 3D 模型之间的鸿沟，实现了自动化的骨骼绑定和面部控制。通过组件级衣物建模（Component-wise Garment Modeling），我们的方法能够生成具有清晰分层、精细纹理且与身体高度对齐的衣物资产。

实验结果表明，该方法在视觉表现和功能实用性上均优于现有的基准模型。尽管我们的系统已经在网格质量和可控性上取得了巨大进步，但未来的工作仍可以进一步探索更复杂的动态服饰模拟以及更丰富的配件交互。总体而言，该框架显著降低了 3D 动漫内容的生产门槛，为交互式娱乐和数字创作提供了新的可能。

# 6. 相关工作

### 2.1 3D 物体生成 (3D OBJECT GENERATION)

近年来，扩散模型在图像生成领域取得了显著突破 。然而，与海量的 2D 图像相比，3D 数据的匮乏使得直接训练具有高度泛化能力的 3D 生成模型充满挑战 。为了缓解这一问题，Poole 等人 (2022) 引入了**分数蒸馏采样 (Score Distillation Sampling, SDS)** 损失函数，利用预训练的 2D 扩散模型来引导 3D 生成 。

虽然基于 SDS 的方法显著提升了 3D 生成的性能，但由于 SDS 本身并不具备强制几何一致性的能力，导致生成的几何形状往往缺乏保真度 。一些方法尝试通过从多视图法线图中重建 3D 几何来解决此问题，但它们可能面临体积一致性有限以及难以处理复杂拓扑结构的问题 。

随着 3D 数据可用性的增加，近期出现的一个趋势是超越基于 SDS 的监督模式 ：

* **基于 LRM（大重建模型）的方法**：通过有限的输入预测三平面（triplane）表示，从而重建 3D 模型 。

* **直接扩散方法**：将扩散模型直接应用于 3D 数据，生成用于 3D 合成的三平面 。

* **表征格式演变**：除了三平面，稀疏体素网格（sparse voxel grids）和基于 VecSet 的表示方法也在 3D 生成中受到关注 。

## 2.2 衣物生成 (GARMENT GENERATION)

**缝制模式 (Sewing patterns)** 已被广泛应用于衣物生成，因为它们符合物理真实感并与现实中的服装结构一致 。这些表示方法提供了清晰的语义结构和高水平的可控性，允许对衣物的合身度、款式和布局进行细粒度调整 。然而：

* 生成精确的模式需要专家知识，且在处理风格化或非常规服装时可能会遇到困难 。

* 由于结构和对齐约束的限制，从图像或文本推断缝制模式仍然是一个挑战 。

除了基于模式的方法外，其他研究还探索了从单目视频重建 3D 动态服装，或利用现有的生成框架来产出服饰 。Luo 等人 (2025) 提出了一种基于原生 3D 扩散的身体对齐可穿戴资产生成方法 。然而，由于衣物与身体之间仍存在细微的穿插现象，生成后通常仍需要后期处理 。

### 2.3 角色生成 (CHARACTER GENERATION)

此前的工作在现实 3D 人体生成方面已取得显著进展 ：

* 
**几何恢复**：通过利用法线图的隐式表示或显式法线积分，重建详细的着装人体表面 。

* **空间优化**：引入双观察空间设计（规范空间与姿态空间）或层级生成策略，以获得更好的服装结构和更高的真实感 。

* **动漫风格探索**：
* Peng 等人 (2024) 使用大重建模型生成动漫角色，但在多方面缺乏细节 。

* He 等人 (2024b) 和 Dong 等人 (2025) 采用**组件化生成策略**来提高整体质量 。

* Qian 等人 (2025) 利用稀疏体素提高生成网格的分辨率，增强了面部和手部等细节区域的质量 。

* **当前局限**：由于在细粒度细节处理上仍有局限，上述所有方法在产出“真正可用”的 3D 动漫角色方面仍然力有不逮 。

### **核心分析总结**

1. **技术演进：** 文献综述展示了从早期的“图像引导生成的 2D 蒸馏（SDS）”向“直接 3D 原生生成（3D Diffusion / LRM）”的转变 。

2. **本文定位：** 作者强调了现有动漫生成方法（如 **CharacterGen** 或 **StdGen** ）在“可用性”上的缺失。这种缺失主要体现在无法平衡“高分辨率细节”与“可直接用于动画的骨骼结构” 。

3. **对齐挑战：** 在衣物生成部分，作者特意指出了“穿模（Penetration）”和“手动后处理”的痛点，这为本文随后提出的 **Body-Aligned（身体对齐）** 策略埋下了伏笔 。

您可以继续阅读 **3 METHOD** 章节，了解他们具体是如何利用 **Anime-SMPL** 来解决 2.3 节中提到的“可用性”难题的。

# 7. 方法
我们的流水线（如图 2 所示）首先根据文本描述或输入图像，生成一张处于规范姿态（例如 A-pose）的角色图像 。基于这张规范姿态图像和 Anime-SMPL 模板，我们估计目标角色的 Anime-SMPL 参数 。与 SMPL 类似，我们使用关节回归器来获取关节位置 。随后，我们在估计出的 Anime-SMPL 网格表面进行点云采样，并使用 VecSet VAE 将其编码为身体潜变量标记 (Body Latent Tokens) 。

这些身体潜变量标记，连同噪声标记、衣物标签标记以及规范姿态图像，一起输入到我们的 MoE 结构多形状 DiT 网络中，以生成 3D 衣物网格 。最后，我们为身体和各个独立的衣物组件生成纹理贴图 。

## 3.1 2D 规范姿态角色生成 (2D CANONICAL POSE CHARACTER GENERATION)
如流水线图所示，我们的框架从文本描述或任意姿态的角色图像中，生成一张处于规范姿态的角色正面图像 。

文本转图像合成 (Text-to-Image)：我们使用成对数据（文本描述及其对应的正面规范姿态图像）微调了 PixArt-Σ 。

图像转图像合成 (Image-to-Image)：类似于 CharacterGen 和 StdGen，我们采用 ReferenceUNet 和 CLIP 编码器作为图像特征提取器，并结合通用 A-pose 骨骼图作为额外条件来合并姿态信息 。

为了训练图像转图像模型，我们在各种相机视角、姿态和面部表情下渲染了多样的动漫角色全视图 。

## 3.2 ANIME-SMPLSMPL 
模型在人体形状估计和 3D 重建中发挥了重要作用，但由于几何和风格上的巨大差异，其在动漫角色上的应用受到限制 。为此，我们引入了 Anime-SMPL，这是一种专为动漫风格角色设计的参数化身体模型 。

Anime-SMPL 参数化：不同于 SMPL 同时参数化姿态和形状，我们的框架专注于形状建模，因为我们的目标是从固定规范姿态的图像中重建角色 。我们对包含 20,000 个角色的数据集进行主成分分析 (PCA)，保留前 98 个主成分来捕捉主要的形状变化 。

关节回归与形状估计：我们使用非负最小二乘法 (NNLS) 估算关节回归矩阵 。我们训练了一个基于 ResNet 的形状预测网络，通过输入图像预测形状参数 $\hat{\beta}$ 。

## 3.3 用于衣物生成的 MoE 结构多形状 DiT (MOE-STRUCTURED MULTI-SHAPE DIT)
合成高质量动漫角色的困难主要源于复杂的几何结构（如多样的发型和复杂的服饰） 。为了应对这些挑战，我们提出了组件化生成策略 。

组件拆解：对于每个角色，我们将除身体外的装扮分解为四个组件：发型、上衣、下装和配饰 。我们采用 VecSet 扩散模型作为核心生成模型来合成这些组件 。

MoE 结构 DiT 块：我们设计了 MoE 结构的多形状 DiT 架构，包含四个 MLP 专家分支，每个分支专门负责一个特定的衣物组件 。除专家分支外，所有参数共享，以最小的参数开销实现组件感知生成 。

身体对齐生成：为了减少穿模伪影，我们将从 Anime-SMPL 表面采样的点云编码为身体潜变量标记 。这些标记与衣物标记连接，为扩散模型提供显式的几何引导，确保衣物与身体几何形状精准对齐 。

## 3.4 纹理生成 (TEXTURE GENERATION)
我们采用 MVAdapter 作为核心组件，将纹理生成分为两个阶段 ：
身体纹理生成：利用 Anime-SMPL 统一的 UV 布局，在 UV 空间直接生成纹理 。扩散模型将身体纹理分解为六个区域：身体皮肤、面部皮肤、左眼、右眼、眉毛和睫毛 。

组件级衣物纹理生成：为了解决不同组件间的偏色/串色 (Color Bleeding) 问题，我们首先训练一个扩散模型将全身图像分解为各个独立衣物组件的放大视图 。这些分割后的图像随后输入 MVAdapter 合成多视角渲染图，最后投影到 3D 表面 。

方法论核心要点总结标准化底座 (Anime-SMPL)：
通过 PCA 定义了属于动漫的“骨骼与肉体”标准，解决了动漫角色比例夸张导致的无法适配传统 SMPL 的问题 。
分而治之 (Component-Wise)：不贪大求全生成整体 Mesh，而是通过 MoE (混合专家系统) 让模型分工合作（头发交给头发专家，衣服交给衣服专家），极大提升了精度 。
几何对齐 (Body-Aligned)：通过将身体的 3D 点云信息“喂”给生成器，让生成的衣服像“贴”在身上一样，从根源上减少了 3D 建模中常见的穿模伪影 。
# 8. 实验
## 4.1 实施细节 (IMPLEMENT DETAILS)
训练：Anime-SMPL 形状预测网络在单张 NVIDIA L20 GPU 上训练，耗时约 4 小时 。
核心模型：MoE 结构多形状 DiT 在 16 张 NVIDIA A100 GPU 上使用 AdamW 优化器训练，学习率为 $1\times10^{-4}$，耗时约 10 天 。
其他模块：包括 2D 规范姿态生成、身体纹理生成及组件级衣物纹理生成在内的其余模块，分别在 8 张 NVIDIA A100 GPU 上独立训练，各耗时约 2 天 。
推理速度：图像生成阶段需 5 秒，Anime-SMPL 参数预测需 2 秒，多形状 DiT 需 40 秒，身体纹理生成需 10 秒，衣物纹理生成需 360 秒 。
数据集：所有模型均在我们的私有数据集上训练，该数据集包含 20,000 个对齐到统一身体模板的动漫角色 。每个角色共享相同的身体网格拓扑（顶点数、顶点顺序及面片连接性完全一致） 。

## 优点 
1. 范式转移：从整体生成到解耦建模组件化哲学：文章摒弃了将角色视为单一网格的传统方法，提出将发型、上衣、下装及配饰进行模块化分解 。这种**解耦生成（Component-wise Generation）**策略不仅符合现代数字资产制作流程（如换装系统），更从本质上降低了单一大型神经网络学习复杂几何拓扑的负担 。
1. 标准化底座的重塑：Anime-SMPL对统计模型的动漫化演进：作者敏锐地发现传统 SMPL 模型在处理动漫夸张比例（如头身比、眼部构造）时的无力感 。通过在 20,000 个动漫样本上进行 PCA 重塑，Anime-SMPL 为动漫 3D 生成提供了一个具备拓扑一致性和原生骨骼的工业级标准底座 。
1. 创新性的几何约束机制Body-Aligned 采样引导：通过从 Anime-SMPL 表面提取点云作为 VecSet 扩散模型的几何先验（Body Latent Tokens），研究有效解决了 3D 生成中臭名昭著的“穿模（Interpenetration）”问题 。这种物理空间上的强约束极大增强了生成结果的稳健性 。
## 缺点
1. 效率瓶颈与实用性矛盾纹理生成耗时过长：实验数据显示，衣物纹理生成（Garment Texture Generation）单次耗时高达 360 秒 。对于一个标榜“自动化”和“交互式媒体应用”的项目 ，长达 6 分钟的等待时间在实时生成或大规模工业管线中几乎是不可接受的。
1. 几何表达的“双刃剑”：SDF 的局限物理模拟不兼容：文章承认通过 Marching Cubes 从 SDF 提取的网格是**双面（Double-sided）**的 。在游戏引擎（如 UE5）的物理布料模拟中，双面薄壳网格往往会导致计算开销翻倍甚至模拟崩溃 。这与其“动画就绪（Animation-ready）”的核心卖点存在逻辑冲突 。
1. 数据集透明度与复现危机闭源数据集依赖：核心性能高度依赖于包含 20,000 个高质量对齐样本的私有数据集 。由于缺乏开源数据集上的定量对比（如 PSNR/SSIM 仅因数据集不同而未予列出），评审很难判定其性能提升究竟源于算法创新，还是仅仅源于高质量私有数据的“算力暴力” 。
1. 复杂场景下的泛化风险姿态规范化的脆弱性：论文坦言 2D 规范姿态生成阶段在面对复杂配饰或极端姿态时会产生扭曲输出 。这意味着整个流水线对初始 2D 分割和姿态纠正的稳健性存在高度依赖。
## 改进方向
从 SDF 到原生网格生成：探索直接在顶点或面片上操作的生成方法（如 MeshTransformer），以消除双面网格伪影并产出更符合美术规范的低多边形拓扑 。

纹理空间的 3D 一致性优化：开发直接在 3D 空间（如纹理图集空间或 3D 潜空间）进行纹理扩散的技术，以取代目前的“投影-多视角合成”方案，彻底解决跨视角不一致和偏色问题 。

实时化加速管线：研究如何利用蒸馏技术（Distillation）或更高效的重建网络（如 LRM 变体）将纹理生成时间缩短至秒级，使其具备集成进交互式引擎（如 UE5 插件）的可能 。

动态物理先验的引入：在生成组件时，不仅考虑静态对齐，还应引入基于物理的厚度属性和材质参数（Material Parameters），使生成的网格原生适配物理引擎的布料计算 。
