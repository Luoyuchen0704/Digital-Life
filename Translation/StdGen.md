# 2. 摘要
我们提出了 StdGEN，这是一种创新的流水线（pipeline），用于从单张图像生成语义分解的高质量 3D 角色，可广泛应用于虚拟现实、游戏和电影制作等领域 。与以往在可分解性受限、质量不理想以及优化时间长等问题上挣扎的方法不同，StdGEN 具有可分解性、有效性及高效性：它能在 3 分钟内生成包含身体、衣服和头发等独立语义组件的精细 3D 角色 。StdGEN 的核心是我们提出的语义感知大重建模型 (S-LRM) 。这是一种基于 Transformer 的泛化模型，能够通过前向推理（feed-forward）从多视角图像中联合重建几何、颜色和语义信息 。我们引入了一种可微分多层语义表面提取方案，用以从 S-LRM 重建的混合隐式场中获取网格 。此外，流水线中还集成了一个专门的高效多视角扩散模型和一个迭代式多层表面精修模块，以促进高质量、可分解的 3D 角色生成 。大量实验表明，我们的方法在 3D 动漫角色生成方面达到了最先进（SOTA）的水平，在几何、纹理和可分解性方面均大幅领先于现有基准 。StdGEN 提供了即插即用的语义分解 3D 角色，并支持灵活的定制化应用。

# 3. 引言
从单张图像生成高质量的 3D 角色在虚拟现实、视频游戏和电影制作等领域有着广泛的应用 。除了自动创建完整的 3D 角色外，市场对生成可分解角色的需求也日益增长，即能够将身体、衣服和头发等不同的语义组件解耦 。这种分解能够显著降低角色编辑、控制和动画制作的难度，从而极大地增强其在各种下游应用中的实用性 。

然而，从单张图像创建此类可分解角色极具挑战性，因为每个组件都可能面临遮挡、歧义以及与其他组件交互不一致等问题 。现有的可分解化身生成方法主要集中在写实风格的着装人体模型上，探索了分层 3D 参数化、显式或隐式表示以及各种优化技术 。这些优化方法通常采用得分蒸馏损失（SDS）来利用 2D 生成先验，这会导致优化时间过长，并产生粗糙且高对比度的纹理 。此外，对 SMPL-X 等参数化人体模型的依赖对于虚拟角色来说是不够的，因为虚拟角色通常具有夸张的身体比例和复杂的服装设计 。

为了解决这些局限性，CharacterGen 被开发出来，旨在利用多视图扩散模型和大重建模型（LRM）从单张图像高效生成角色 。尽管 CharacterGen 在处理各种姿态的图像时展示了令人印象深刻的生成能力，但它只能产生不可分解的整体水密网格（watertight meshes） 。这些网格需要大量的后期人工劳动来拆分、编辑或制作动画，限制了其应用范围 。此外，生成的网格质量往往不尽如人意，特别是在角色面部和衣物等细节方面 。因此，高效生成高质量、可分解的 3D 角色仍然是一个悬而未决的挑战 。

在这项工作中，我们提出了 StdGEN，这是一种高效且有效的流水线，用于从任意姿态的单张图像生成高质量、可分解的 A-pose 3D 角色 。其核心是一个新型的语义感知大重建模型 (S-LRM)，它创新性地在原始大重建模型中引入了语义属性，从而实现了几何、颜色和语义场的统一高效重建 。此外，我们提出了一种可微分的多层语义表面提取方案，用于从重建的隐式场中提取分解后的语义 3D 表面，支持与显式网格进行端到端的联合训练 。最终，S-LRM 能够以全前向推理的方式，根据多视图输入图像高效生成语义分解、完整且一致的 3D 表面 。

基于 S-LRM，StdGEN 包含三个关键阶段：多视图扩散、前向重建和网格精修，每个阶段都经过技术创新以实现有效的分解 。具体而言，给定一张任意姿态的单张参考图像，专门的高效多视图扩散模型首先生成不同摄像机视角下的多个 A-pose RGB 图像和法线贴图 。接着，这些生成的图像由我们的 S-LRM 处理，以全前向的方式重建几何、颜色和语义信息 。在获得初步的分解表面生成后，我们通过提出的迭代式多层网格精修方法，利用扩散生成的 2D 图像和法线贴图作为引导，进一步提升网格质量 。因此，StdGEN 可以在几分钟内从单张参考图像生成高质量的可分解 3D 角色 。

为了促进语义分解的训练和测试，我们基于先前工作进一步开发了 Anime3D++ 数据集，重点关注因互联网资源丰富而广受关注的动漫角色，并提供了精细标注的多视图、多姿态语义部件 。我们的实验表明，StdGEN 超越了所有现有基准方法，在任意姿态和 A-pose 的 3D 角色生成方面均达到了最先进的性能 。此外，StdGEN 的可分解生成能力实现了灵活的定制化，能够造福众多的下游应用 。
# 4. 结论
在本文中，我们介绍了 StdGEN，这是一个创新的流水线，用于从单张图像生成语义分解的高质量 3D 角色 。我们提出了一种新型的语义感知大重建模型 (S-LRM)，用以联合重建几何、颜色和语义信息 。配合高效的 2D 扩散模型和迭代式多层精修模块，该系统能够从任意姿态的单张图像中实现高质量的生成 。StdGEN 在几何、纹理和可分解性方面均表现出优于现有基准方法的性能，并能提供即插即用、语义分解且可定制的 3D 角色，展示了在多种应用场景下的巨大潜力 。

# 6. 相关工作
## 2.1. 3D 生成 (3D Generation)
为了规避训练期间对大量 3D 资产的需求，一些方法建议利用强大的 2D 预训练扩散模型进行 3D 生成 。早期工作通过得分蒸馏采样 (SDS) 结合预训练 2D 扩散模型进行概率密度蒸馏 。这些方法利用体渲染逐渐优化随机初始化的辐射场，使得生成单个物体非常耗时 。随后的研究继续增强 3D 内容生成的细节与准确性，并进一步探索不同的应用场景 。然而，仅依赖 2D 先验进行 3D 生成往往会导致几何表示较差（例如多面 Janus 问题），因为通过文本提示词精确控制视角具有挑战性 。大规模 3D 数据集（如 Objaverse）的出现，使得向模型引入 3D 先验成为可能 。

一些工作利用视角一致的图像来微调扩散模型 。Zero-1-to-3 通过微调预训练模型进行新视角合成 (NVS)，将 3D 先验集成到 2D 稳定扩散模型中 。为了进一步增强多视图一致性，近期一些工作提出在单个生成过程中同步生成多视图图像，并通过注意力机制中的特征交互实现 3D 空间约束 。此外，3D 原生生成方法展示了强大的几何生成能力 。然而，这些方法遵循指令的能力通常较弱；因此，在需要精确还原参考图像的场景（如 3D 角色生成）中，它们面临实现预期结果的挑战 。

## 2.2. 大重建模型 (Large Reconstruction Model)大重建模型 (LRM) 
利用基于 Transformer 的模型将单张图像特征映射到隐式三平面 (Tri-plane) 表示 。Instant3D 通过输入多视图图像而非单张图像扩展了 LRM 。LGM、GRM 和 GS-LRM 将 3D 表示替换为 3D 高斯泼溅 (3D Gaussians)，利用其在渲染效率和低显存占用方面的优势 。InstantMesh 和 CRM 通过为生成流水线配备 FlexiCubes 显式建模几何，实现了高质量的表面提取和极快的渲染速度 。后续工作进一步探索应用先进的模型架构或 3D 表示，旨在提高重建的效率、真实感和泛化能力 。

通过与多视图扩散模型集成，这些 LRM 可以实现文本到 3D 或单张图像到 3D 的生成 。然而，所有这些方法通常产生的是整体模型 。相比之下，我们的方法生成语义分解的角色，使得编辑和动画等下游处理更加高效 。

## 2.3. 3D 角色生成 (3D Character Generation)
由于高精度需求和数据稀缺，3D 角色生成是一个具有挑战性的问题 。一类工作利用 3D 感知 GAN 来建模数字人的分布 。近期，基于 SDS 的方法展示了生成各种风格化角色的可能性，但面临优化时间长和风格精确控制困难的问题 。Frankenstein 专注于根据 2D 布局生成分解的、无纹理的 3D 网格，这限制了从参考图像实现高保真重建的潜力 。

CharacterGen 通过图像条件多视图扩散模型将输入姿态校准为规范的多视图图像，随后利用 LRM 进行 3D 角色重建和多视图纹理反向投影，但其几何和纹理质量仍然有限 。相比之下，我们的方法采用了语义感知的全前向（feed-forward）范式，仅通过对任意参考图像的一次前向传递即可生成高质量、可分解的角色，提供了显著的效率和质量提升 。
# 7. 方法


StdGEN 的流程首先从参考图像生成**多视图规范化角色**开始（第 3.1 节）。为了从多视图图像中重建出可分解的 3D 角色，我们通过增加**语义场**扩展了大重建模型（LRM），从而实现基于语义的分层生成（第 3.2 节）。最后，设计了一个**多级精修过程**来增强结果，改善几何结构并提供更详细的纹理（第 3.3 节）。StdGEN 流水线的概述如图 2 所示 。

### 3.1. 

多视图生成与规范化 (Multi-view Generation and Canonicalization) 

给定一张参考角色图像，我们的目标是生成一组保持 3D 一致性的多视图图像及相应的法线贴图 。该过程包含两个步骤：将任意姿态的参考图像**规范化**为 A-pose 角色，并从 A-pose 表示生成多视图 RGB 和法线 。

* **任意参考到 A-pose 的转换**：直接重建任意姿态的 3D 角色模型会受到不同视角下自遮挡的影响 。因此，我们设计首先在 2D 空间进行角色规范化 。我们选择 **A-pose** 作为目标表示，因为它更有利于后续的多视图生成、重建以及骨骼绑定等应用 。参考前人工作，我们采用了增强了 ReferenceNet 的 Stable Diffusion 模型，将 2D 任意姿态参考图转换为 A-pose 图像，ReferenceNet 有助于保持图像中的身份特征 。

* **多视图 RGB 与法线生成**：此阶段为后续的 S-LRM 提供全面信息 。具有多视图一致性的法线对于在网格精修中捕捉丰富的表面细节至关重要 。我们改进了 Era3D，通过在视图间以及 RGB 与法线贴图之间使用显存高效的行向注意力机制，同时生成高分辨率（最终达到 1024）的 RGB 和法线，确保了精细的空间一致性 。该阶段以单张 A-pose 图像为输入，产出六个视角的正交投影 。

### 3.2. 

语义感知大重建模型 (Semantic-aware Large Reconstruction Model) 

与重建整体 3D 网格的现有模型不同，我们的目标是生成具有**分解组件**（包括极简着装人体、外部服饰和头发）的角色，使其在视觉准确的同时，在 3D 游戏和动画流水线中具备功能通用性 。为此，我们提出了 **S-LRM**，它扩展了 NeRF/SDF，以便以全前向方式同时编码语义、外观和几何信息 。

* 
**语义等效 NeRF 与 SDF**：我们将 NeRF 扩展了语义场，模型 $F_{\theta}$ 输出颜色 $c$、密度 $\sigma$ 和语义分布 $s$ 。

* 在渲染每像素颜色时，通过整合采样点的语义概率 $p_{s,i}$，可以计算特定语义下的等效像素颜色 $\tilde{C}_{s}(r)$ 。

* 对于 **SDF** 提取，由于其没有透明度概念，我们根据以下原则构建**等效 SDF**：原始 SDF 的零值作为硬约束；当原始 SDF 为正时，等效值也应为正；在相关语义转换的边界处，等效 SDF 应为零 。

* **等效 SDF 计算公式**：

$$f_{i,s}=max(f_{i},(max_{r\ne s}p_{i,r})-p_{i,s})$$

这种表述确保了特定语义的正确分解，并与 FlexiCubes 网格提取完全兼容 。

* **模型结构与监督**：S-LRM 核心结构衍生自 InstantMesh，包含 ViT 编码器、图像到三平面转换器和特征解码器，并关键性地增加了**语义解码器** 。为了学习内部结构和被遮挡的信息，我们采用了**多层语义监督策略** ：

* **第一阶段**：在具有单层语义的 NeRF 上训练 。

* **第二阶段**：在具有多层语义的 NeRF 上训练，根据空间关系从外到内分层监督，获取内部颜色与语义 。

* **第三阶段**：在具有多层语义的**网格 (Mesh)** 表示上训练，利用 FlexiCubes 实现高效的高分辨率训练 。

### 3.3. 

多层精修 (Multi-layer Refinement) 

鉴于大重建模型在几何精度上的局限性，我们提出了一种**迭代优化机制**进行多层网格精修 。

* **分阶段优化**：首先提取并仅优化**基础人体模型**；完成后叠加**衣服网格**并仅优化衣服部分；最后加入**头发网格**并仅优化头发 。

* **引导与损失**：优化过程由扩散模型生成的**多视图法线贴图**引导 。我们引入了**碰撞损失 (Collision Loss)** 以确保外层网格始终位于内层网格的法线方向外侧，防止穿模 。最后，通过颜色反向投影完成上色 。

# 8. 实验
## 4.1. Anime3D++ 数据集 (Anime3D++ Dataset)
为了满足质量和可分解性的要求，我们基于原始的 Anime3D 数据集开发了 Anime3D++ 数据集 。我们最初从 VRoid-Hub 收集了约 14,000 个 3D 动漫角色模型，随后筛选出 10,811 个高质量模型，并统一标准化为手臂下倾 45° 的 A-pose 。渲染内容包括 RGB 图像、深度图、法线贴图以及用于分层监督的语义图 。每个角色都分别渲染了完整模型、仅基础极简人体模型，以及基础人体加衣服的模型 。

## 4.2. 结果与对比 (Results and Comparisons)
鉴于现有方法无法从单张任意角色图像生成分层 3D 模型，我们在 Anime3D++ 数据集的测试集上将其非分层生成结果与其他方法进行了对比 。为了确保公平性，我们解耦了姿态规范化组件，针对任意姿态和 A-pose 的参考输入分别进行了 2D 和 3D 对比 。

定量结果 (Quantitative Results)：2D 多视图生成：我们将本方法与 Zero-1-to-3、SyncDreamer、Era3D 和 CharacterGen 进行了对比 。3D 角色生成：对比对象包括基于 SDS 优化的 Magic123、ImageDream；前向推理方法 OpenLRM、LGM、InstantMesh；以及直接网格重建的 Unique3D 和 CharacterGen 。如表 1 所示，我们的方法在标准姿态和任意姿态设置下均表现更优 。现有的 2D 方法往往难以保留足够的几何和外观信息，而 SDS 方法通常存在几何模糊和 Janus 问题 。Unique3D 虽然指标较高，但受限于不稳定的网格初始化 。相比之下，我们的方法在所有情况下都一致地取得了卓越结果 。

定性结果 (Qualitative Results)：图 5 展示了不同方法的局限性：InstantMesh 受限于网格分辨率，纹理精度不足；Unique3D 容易在深度估计不准时发生几何坍塌；CharacterGen 则经常产生视觉干扰明显的黑色伪影 。我们的方法展现了更优的几何准确性和纹理保真度 。

分解结果 (Decomposed Results)：图 6 展示了我们的方法从单张参考图重建分解角色的能力 。重建的网格展现了极高的几何精度，同时语义分解也非常准确，成功解耦了基础人体、衣服和头发 。衣服的截面视图显示其内部完全是空心的，这显著增强了下游物理仿真和绑定的潜力 。

用户研究 (User Study)：我们邀请了 28 名志愿者对 16 组随机挑选的结果进行评估 。如表 2 所示，我们的方法在整体质量、保真度、几何质量和纹理质量这四个维度上均被选为最优，且胜出率极高（约 75%-82%） 。

## 4.3. 应用 (Applications)
我们的流水线仅需文字提示词、掩码和 2D 域的局部重绘扩散模型即可实现多样化的 3D 编辑 。如图 7 所示，该框架允许用户轻松进行角色定制，包括更换服装和修改发型 。通过将 2D 修改转化为 3D 重建，我们在多层精修阶段选择性地替换相关网格组件，同时保留未修改的部分 。这弥合了 2D 图像编辑与 3D 角色定制之间的鸿沟 。
# 9. 评论
## 一、 研究优点 (Strengths)
极佳的工业落地价值：不同于以往生成“一大块”不可编辑网格的方法 ，该研究生成的角色在物理结构上是合理的（身体、衣服、头发分离且内部中空），这直接对接了游戏引擎中的碰撞检测和物理仿真需求 。

创新的语义提取方案：提出的“语义等效 SDF”公式 逻辑严密，它没有简单地为每个类别分配解码器，而是通过一套竞争机制从统一场中提取表面，保证了各组件在拓扑上的严密衔接 。效率与质量的平衡：在 3 分钟内完成全流水线生成（含精修） ，相比于动辄数小时的 SDS 优化类方法 ，具有显著的工程优势。数据集贡献：开发的 Anime3D++ 数据集提供了高质量的多层标注，为该细分领域提供了必要的基准 。

## 二、 核心缺陷审查 (Strict Scrutiny: Weaknesses)
严重的类别局限性与泛化性存疑：
缺陷：模型高度依赖于 Anime3D++ 动漫数据集 。
分析：摘要声称“支持广泛应用” ，但实验主要集中在动漫角色 。作者承认在非动漫风格（如写实风格）上存在严重的“风格偏见”（如脸部变尖） 。
质疑：如果该方法在处理写实人体或非人类生物时表现崩溃，那么其“通用框架”的叙事就显得言过其实 。两阶段生成导致的误差累积：缺陷：流水线严重依赖第一阶段的“规范化扩散”结果 。
分析：如果 2D 扩散模型生成的 A-pose 图像出现肢体缺失或身份特征漂移（Identity Drift），后续的 S-LRM 无法纠正这些底层错误 。
质疑：实验中缺乏对“规范化失败案例”的鲁棒性分析。分辨率瓶颈与内存权衡：
缺陷：核心 S-LRM 输出的三平面分辨率仅为 $64\times64$ 。
分析：虽然有后期精修，但基础几何的精度上限被锁死在了这个极低的分辨率上 。
质疑：精修阶段过度依赖法线贴图的纹理欺骗，是否掩盖了基础网格在精细解剖结构上的贫瘠？ “多层精修”的盲目性：缺陷：精修过程是分阶段迭代的（先人体再衣服再头发） 。分析：这种单向流水线缺乏反馈。如果基础人体的精修过度膨胀，可能会导致衣服层的几何优化失去空间 。

## 三、 改进建议 (Actionable Improvements)
引入多模态大模型引导的语义分割：不要仅限于现有的固定类别。建议集成 SAM (Segment Anything Model) 或类似的开放词汇分割模型，以支持更细粒度的分解（如外套、内衬、饰品的分离），从而提升通用性 。

端到端的一致性约束：增加一个从 3D 重建结果反向投影回原始“非规范化”姿态的重构损失函数。这能强制模型在规范化的同时，确保身份特征（Identity）的一致性，减少两阶段间的信息损失 。

提升三平面分辨率或引入稀疏表征：$64\times64$ 的分辨率在 2026 年的学术标准下已显落后 。建议尝试使用八叉树结构或稀疏卷机提升底层表征的分辨率，从根本上减少对后期精修（Refinement）的过度依赖 。

强化穿模（Interpenetration）处理逻辑：目前的碰撞损失 $?_{col}$ 较为初级。建议引入基于物理的软约束（Soft-body constraints），确保在极端姿态下，衣服与皮肤的间隙依然符合解剖学常识 。
